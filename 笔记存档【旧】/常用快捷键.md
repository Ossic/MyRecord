# 常用键盘快捷键

后退 

>Alt + ←

前进 

>Alt + →

向上/上移 

>Alt + ↑

重命名 

>F2

打开资源管理器 

>win + E

删除到回收站 

>ctrl + D

彻底删除 

>shift + delete

到行头/当前窗口顶部 

>HOME

到行尾/当前窗口底部 

>END

切换程序 

>Alt + Tab

切换标签(主要浏览器) 

>ctrl + Tab

倒序切换标签 

>ctrl + shift + Tab

最小化

> win + M

还原最小化的窗口 

>WIN+ Shift + M 

显示桌面 

>win + D

右键菜单 

>shift + F10

移动的任务栏 

>win + B 

显示属性 

>Alt + enter(回车)

最大化/最小化/靠左/靠右 

>win + ↑/↓/←/→

调出菜单 

>Alt + 空格

	+C 关闭
	
	+N 最小化
	
当前窗口或文档的最上端 

>Ctrl + Home 

当前窗口或文档的最下端 

>Ctrl + End
